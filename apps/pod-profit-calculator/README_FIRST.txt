 POD Profit Calculator Pro
 README FIRST
========================================

Thank you for choosing POD Profit Calculator Pro! This tool helps print‑on‑demand sellers quickly calculate profits, compare supplier/channel combinations, and manage product data — all in one self‑contained file.

WHAT YOU GET
------------
• POD_Profit_Calculator.html  – The complete application (open in any modern browser).

No installation, no server, no internet required after loading. All data stays on your device.

QUICK START
-----------
1. Double‑click POD_Profit_Calculator.html to open it in Chrome, Edge, Firefox, or Safari.
2. The Calculator view is ready to go. Fill in the product details and see results instantly.
3. Save products, compare scenarios, explore analytics, and export data as needed.

For a full walkthrough, open the Help view inside the app (nav item 06) or read the included User Guide (USER_GUIDE.txt).

SYSTEM REQUIREMENTS
-------------------
• A modern web browser (Chrome 90+, Edge 90+, Firefox 90+, Safari 15+).
• JavaScript and localStorage enabled (default in all modern browsers).

PRIVACY & DATA
--------------
All product data and settings are stored locally in your browser using localStorage. Nothing is ever sent to the internet.
