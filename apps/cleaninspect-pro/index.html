Quick Start
  1. Add your clients in the Clients view
  2. Customize the 6 default templates or add your own
  3. Start a new inspection (top-right button or ⌘N)
  4. Mark each item as Pass, Fail, or N/A
  5. Add notes for any failed items
  6. Complete the inspection and print the professional PDF
