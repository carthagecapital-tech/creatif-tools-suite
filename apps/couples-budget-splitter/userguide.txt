1. DASHBOARD OVERVIEW
--------------------------------
The Dashboard gives you a snapshot of your finances:
  • Total spent this month
  • Partner balance (who owes who)
  • Shared savings progress
  • Debt settlement card
  • Recent activity (last 5 expenses)
  • Summary of monthly subscriptions, goals, and transactions

Use this for a quick check before a money talk.

--------------------------------
2. EXPENSES & SPLITS
--------------------------------
Go to Expenses to add, edit, or delete shared expenses.

SPLIT METHODS EXPLAINED
  • 50/50 Even – simplest method, both pay the same.
  • Income-Based – share proportional to your incomes.
    (Set your incomes in Settings first.)
  • Custom Percentage – drag the slider to choose exactly.
  • Fixed Amount – each partner pays a set amount; the two
    must add up to the total.

You can also add notes, change the date, and pick a category.

SEARCH & FILTER
  Use the search box (Ctrl/Cmd+K) to find expenses by name
  or category. Click the filter chips to show only one
  category.

SORT & PAGINATION
  Click column headers (Date, Name, Amount) to sort.
  If you have many expenses, use the arrow buttons to go
  through pages.

--------------------------------
3. SUBSCRIPTIONS
--------------------------------
Track Netflix, rent, gym, etc. Enter:
  • Name
  • Cost per billing cycle
  • Who pays
  • Billing cycle (monthly, quarterly, yearly)
  • Category

The app calculates the monthly equivalent and always splits
subscriptions 50/50.

--------------------------------
4. SAVINGS GOALS
--------------------------------
Create shared goals (e.g., "Hawaii Vacation $5000").
Set a target amount and optional date. Update the current
amount as you save. The progress bar shows how far you've come.

--------------------------------
5. ANALYTICS
--------------------------------
See charts for:
  • Spending by category
  • Monthly trends (last 6 months)
  • Contribution comparison (paid vs owed)
  • Split type usage

Use these to spot where your money is going and adjust.

--------------------------------
6. SETTINGS
--------------------------------
Customise:
  • Partner names
  • Monthly incomes (affects income-based splits)
  • Default split type
  • Currency (11 currencies supported)
  • Accent colour (drag the slider)
  • Dark mode (click the moon/sun icon in the sidebar)

All changes save automatically. You’ll see a brief "Saved"
indicator in the footer.

--------------------------------
7. IMPORT & EXPORT
--------------------------------
Backup your data:
  • Export All Data – saves expenses, subscriptions, and goals.
  • Export Expenses Only – just expenses.
  • Import Expenses – restore expenses from a CSV file
    (max file size 5 MB).

Use this to move your data to a new computer or share between
partners.

--------------------------------
8. TIPS FOR COUPLES
--------------------------------
  • Log expenses right away – keeps balances accurate.
  • Review monthly – check the Analytics tab.
  • Set goals together – it’s motivating!
  • Talk about big expenses before adding them.
  • Use income‑based splits if one partner earns more.
  • Export regularly – back up your data.
  • Print a summary (Ctrl/Cmd+P) for your money meeting.

KEYBOARD SHORTCUTS
  Ctrl/Cmd+K – focus search box
  Ctrl/Cmd+P – print
  Escape – close any open dialog
